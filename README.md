This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## How to use
1. Add Square adds a new square
2. You can edit the square by clicking on it
3. You can change the colors
4. Clicking on clear button will clear out the canvas
5. Clicking on Preview will toggle to preview mode
